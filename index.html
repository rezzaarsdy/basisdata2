BASIS DATA II


Kuliah ini merupakan kelanjutan dari mata kuliah Basis Data I Fokus utama pada mata kuliah ini adalah pada sistem Manajemen Basis Data Teori dan Konsep yang akan dikuasai :

  • Komponen Utama Sistem Manajement Basis Data yang meliputi query optimizer, query executor, storage manager, access methods dan transaction processor
  • Pendekatan Umum untuk penyimpanan dan pemrosesan data dengan volume sangat besar
  • Data independece dalam sistem basis data
  • Fasilitas yang disediakan oleh sistem basis data untuk mendukung format data seperti stream
